# Список всех ссылок на Telegram в проекте

## 1. Прямая ссылка на канал/группу поддержки

**`https://t.me/snovidec_ru`**

Используется для:
- Ссылки "Служба поддержки" в меню (авторизованные и неавторизованные пользователи)
- Кнопки поддержки на страницах ошибок
- Ссылки в статьях-инструкциях (категории "Толкование снов" и "Техническая поддержка")
- Ссылки в статьях-инструкциях (категория "Безопасность и приватность" - восстановление пароля и удаление аккаунта)

## 2. Ссылка для шаринга через Telegram

**`https://t.me/share/url?url={URL}&text={TEXT}`**

Используется в JavaScript функциях для шаринга:
- Результатов толкования снов (`dream-analyzer/show.blade.php`)
- Результатов анализа отчетов (`reports/analysis.blade.php`)

Формат: динамическая ссылка, где:
- `{URL}` - URL страницы для шаринга (кодируется через `encodeURIComponent`)
- `{TEXT}` - текст для шаринга (кодируется через `encodeURIComponent`)

## Итого:

В проекте используется **2 типа ссылок на Telegram**:

1. **`https://t.me/snovidec_ru`** - прямая ссылка на канал/группу поддержки (используется везде для связи с поддержкой)
2. **`https://t.me/share/url?url=...&text=...`** - ссылка для шаринга контента через Telegram (используется в функциях шаринга)
