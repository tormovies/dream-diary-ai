## Процесс деплоя

1. `git pull origin main` - получить код
2. `npm install && npm run build` - пересобрать фронтенд
3. `composer install` - обновить PHP зависимости
4. `artisan migrate` - применить миграции
5. Очистить и пересоздать кэши
